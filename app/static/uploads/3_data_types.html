1. What are Data Types?

Data types tell Python what kind of value a variable holds.
Python is dynamically typed ‚Üí you don't need to declare the type!

Most common built-in data types in Python (2025‚Äì2026):

Type Category Mutable? Example Real-life meaning
int Number No 42, -7, 0 Whole numbers, age, quantity
float Number No 3.14, -0.001, 99.99 Decimal numbers, price, temperature
str Text No "Hello", 'Python', "" Names, messages, sentences
bool Logical No True, False Yes/No, on/off, correct/wrong
NoneType Special No None Nothing, no value, placeholder

2. Main Data Types ‚Äì Quick Overview

int ‚Üí Integer (Whole Numbers)

Unlimited size in Python 3! (no 2¬≥¬π-1 limit like old languages)

age = 15 students_count = 28 negative = -100 very_big = 999999999999999999999 print(type(age)) #
float ‚Üí Floating Point (Decimal Numbers)

Used for precise measurements, money (sometimes), science

pi = 3.14159 temperature = -2.5 price = 1499.99 print(type(temperature)) #
str ‚Üí String (Text)

Anything inside " " or ' '

name = "Nowsad" message = 'I love Python!' empty = "" print(type(name)) #

Note: Strings are immutable (cannot change single character directly)

bool ‚Üí Boolean (True / False)

Very important for conditions & decisions

is_student = True has_passed = False adult = age >= 18 # will be False if age=15 print(type(is_student)) #
NoneType ‚Üí None

Represents absence of value

result = None username = None # no one logged in yet print(type(None)) #

3. How to Check Data Type?

x = 100 y = 3.14 name = "Sara" active = True nothing = None print(type(x)) # print(type(y)) # print(type(name)) # print(type(active)) # print(type(nothing)) #
type( ) is your best friend when learning!
Always use it when you are confused about what something is!
