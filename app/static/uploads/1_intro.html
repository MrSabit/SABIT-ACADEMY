1. What is Python?

Python is a high-level, interpreted, general-purpose programming language created by Guido van Rossum and first released in 1991.

Key Features of Python

  • Very easy to read and easy to learn (beautiful syntax)
  • Interpreted ‚Üí No compilation needed, run code directly
  • Dynamically typed ‚Üí No need to declare variable types
  • Multi-paradigm ‚Üí Supports object-oriented, procedural and functional programming
  • Huge community & tons of libraries (pip install anything!)
  • Cross-platform (Windows / macOS / Linux / even phones & servers)

Where is Python used today? (2025‚Äì2026)

  • Web development (Django, FastAPI, Flask)
  • Data Science & Machine Learning (pandas, numpy, tensorflow, pytorch)
  • Automation / Scripting
  • AI / Generative AI tools
  • Game development (pygame, godot python support)
  • DevOps & Cloud (Ansible, AWS boto3, kubernetes operators)
  • Cybersecurity tools

3. Comments in Python

Comments are lines that Python ignores.
They are written only for humans to understand the code.

Types of Comments

Type Symbol Example Usage
Single line # # This is a comment Most common
Multi-line (kind of) """ ... """ or ''' ... '''
