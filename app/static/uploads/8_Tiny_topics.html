1. Escape Sequences (Special Characters)

Backslash \ tells Python: "the next character is special!"
Escape Meaning Example Output
\nNew line"Hello\nWorld"Hello
World
\tTab (4 spaces)"Name:\tRahim"Name: Rahim
\\Literal backslash"C:\\Windows"C:\Windows
\"Double quote inside string"He said \"Hi\""He said "Hi"
\'Single quote inside string'It\'s me'It's me
print("Hello\nWorld") # new line print("Name:\tAge:\tCity") # columns print("He said \"Python is fun!\"") # quote inside print('It\'s a beautiful day') # apostrophe print("Path: C:\\Users\\Miftahul") # real windows path

2. Raw Strings ‚Äì Ignore Escapes

Put r before string ‚Üí backslashes are treated literally
normal = "C:\new\test" # \n and \t are interpreted! raw = r"C:\new\test" # exactly as written print(normal) # C: est (because \n = new line, \t = tab) print(raw) # C:\new\test
Very useful for: file paths, regular expressions

3. Multi-line Strings (Triple Quotes)

message = """ Dear Student, Welcome to Python! We hope you enjoy the journey. Best regards, Teacher """ print(message) # Also useful for docstrings (function documentation) def add(a, b): """This function adds two numbers. Example: add(5, 3) ‚Üí 8""" return a + b

4. f-strings ‚Äì Modern & Easiest String Formatting

Fastest & most readable way in Python 3.6+
name = "Sara" age = 14 height = 5.3 print(f"My name is {name}") print(f"I am {age} years old") print(f"Next year I will be {age + 1}") print(f"Height: {height:.1f} feet") # 1 decimal place print(f"Hi {name.upper()}, welcome!")

5. None vs Empty vs Zero

print(None) # nothing print("") # empty string print(0) # zero number print([]) # empty list # Very important checks: if None: print("This won't print") if "": print("This won't print") if 0: print("This won't print") if []: print("This won't print") # But this IS True: if " ": print("Space is NOT empty!") # prints

6. pass ‚Äì Do Nothing Statement

# Useful when you need empty block (Python requires at least one line) if age < 18: pass # "I'll write this code later" else: print("Adult") # Also common in classes/functions you plan to implement later def future_feature(): pass
